# rollup-lib-starter

基于 rollup 搭建的 library 脚手架，支持ES6、TypeScript、Sass。同时支持多种模块模式的输出。

## 快速开始

```bash
git clone
cd rollup-starter-lib
npm install
```

## 打包

```bash
npm run build

# output:
# dist/demo.cjs.js
# dist/demo.esm.js
# dist/demo.umd.js
```

## 测试

```bash
npm test
```
